1. Package Layout
```text

```

```text

```
2. build.sh Requirements

- Must define:
 - TERMUX_PKG_HOMEPAGE
 - TERMUX_PKG_DESCRIPTION
 - TERMUX_PKG_LICENSE
 - TERMUX_PKG_VERSION
- Must NOT:
 - Download prebuilt binaries without verification
 - Use hardcoded paths outside $PREFIX
3. Validate Locally
